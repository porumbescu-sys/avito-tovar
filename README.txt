Мой Товар — Streamlit-версия

Что загрузить в GitHub:
- streamlit_app.py
- requirements.txt
- папку .streamlit с файлом config.toml

Как задеплоить:
1. Создайте репозиторий GitHub.
2. Загрузите туда эти файлы.
3. На streamlit.app выберите репозиторий.
4. Main file path: streamlit_app.py
5. Deploy.

Что умеет:
- загрузка Excel/CSV
- правка цен прямо в интерфейсе
- поиск по артикулам и связанным кодам из названия
- шаблон 1 и шаблон 2
- выбор цены: -12%, -20%, своя скидка
- округление вверх до 100
